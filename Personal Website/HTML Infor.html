Formatting

Bold:

<b>This is a <b> tag, used for bold text.<b>

<i>This is an <i> tag, used for italicized text.<i>

<u>This is an <u> tag, used for underlined text.<u>

<p style="color:red">This is a CSS colour tag, used to change the colour of text from within HTML.<p>

<p style="font-family:verdana">This is a CSS font tag, used to change the font of text.<p>

<p style="font-size:24px">This is a CSS size tag, used to change the size of text.<p>


Tables

Tables are formed by using the <table> tag.
Below is an example of a simple table:

<table border="1px">
<tr>
<th>This is a part of a table.</th>
<th>To separate columns, add a <th> tag!</th> <th>To separate columns, add a <th> tag!</th>
<th>Just like this~<th>
</tr>
<tr>
<th>To separate rows, add a <tr> tag.</th>
<th>This creates a full table with columns and rowa.</th>
<th>Below you can see the output of this section of code:</th>
</tr>
</table>
